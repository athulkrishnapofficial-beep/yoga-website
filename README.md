## ‚öôÔ∏è Installation & Setup

To get a local copy up and running for development or contribution, follow these simple steps.

### Prerequisites

  * Node.js (LTS version recommended)
  * npm or yarn

### Steps

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/athulkrishnapofficial-beep/yoga-website.git
    cd yoga-website
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    # or
    # yarn install
    ```
3.  **Run the development server:**
    ```bash
    npm run dev
    # or
    # yarn dev
    ```
    The application should now be running at `http://localhost:5173` (or a similar port).

## üî® Build for Production

To create a production-ready, optimized build of the application:

```bash
npm run build
# or
# yarn build
```

This command bundles the application into the `dist` folder, which is ready to be deployed to any static host (like Vercel, Netlify, or GitHub Pages).
